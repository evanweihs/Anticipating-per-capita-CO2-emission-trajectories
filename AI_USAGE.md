# AI Usage Declaration

Artificial intelligence tool (ChatGPT) was used in this project as **technical and editorial support**. In particular, AI assistance contributed to:

- resolving specific coding issues and debugging problems encountered during development;
- helping restructure an initially exploratory Jupyter Notebook into a clean, modular, and reproducible Python codebase suitable for a GitHub repository;
- supporting the formatting and presentation of the final report, including guidance on LaTeX structure and academic writing conventions.
